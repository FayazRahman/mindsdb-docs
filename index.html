    Install MindsDB

    MindsDB is an open-source AI layer for existing databases that allows you to effortlessly develop, train and deploy state-of-the-art machine learning models using SQL queries.

    AutoML in Databases

    Learn how to run Machine Learning models as Tables straight in the most popular databases using SQL. Follow detailed tutorials from install to interpreting results with Explainable AI features.

    AutoML in Python

    Explore how to create, train and test neural-networks-based Machine Learning models with no complex data science knowledge required. You have options to customize the model or query it in a fully automated way using a single query.

    AutoML Visualization Toolkit

    Run AutoML models using an intuitive graphical user interface with almost no setup and coding overhead. Easily clean & validate your datasets, train AutoML models and run predictions using cloud-based (or local) server infrastructure. See how the model reached its conclusions with intuitive visualizations of Explainability Features.

    AutoML API

    Need Machine Learning in your favourite programming language? No problem. MindsDB Server allows you to leverage all AutoML and explainability capabilities of MindsDB Stack using powerful REST API's.

    AutoML SDK

    Build custom solutions with Machine Learning capabilities with a full control over your data using MindsDB Server. Automatically train neural-networks-based ML models and make predictions from your own software and provide explainable AI features to your business users using JavaScript or Python SDK.
